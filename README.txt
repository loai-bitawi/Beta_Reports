This project is a course project that aims to develop a GUI for the user to
connect to MYSQL database, explore data and extract data reports with data manipulation tools and data plots

Main Features: 
1.	Ability to deploy any database in case it is well-structured in terms of primary and foreign keys.
2.	Descriptive Statistics.
3.	Table View with additional data (Data Report builder).
4.	Charts (Line, Bar, Scatter, and pie).
5.	Sorting and Aggregations.
6.	Table automatic joins up to 3 tables. (no need to define relations, detected automatically)
7. 	Simple and responsive GUI

the file named Beta Reports describes all project features in details with snapshots.
The file named Database_sample.sql is an SQL dump that can be imported to MYSQL database as a sample database
